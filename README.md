### 如何运行我的项目

1. github项目访问：https://github.com/flzzzp/-flzzzp-.github.io.git

2. 拉取最新代码：

   ```bash
   git pull
   ```

   重新安装依赖：

   ```bash
   cd backend
   npm install
   ```

3. 下载并配置MySQL数据库：https://dev.mysql.com/downloads/mysql/。 下载mysql2数据库驱动；

4. 下载并配置node.js跨平台运行环境；

5. 解压后打开backend文件夹终端，输入下列命令启动服务器：

```shell
node server-mysql.js
```

5. 浏览器中搜索本地访问那一行的链接，即可正常使用。
